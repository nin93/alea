    class Alea::MT19937

    • Alea::MT19937
    • Reference
    • Object

    Overview

    MT19937 engine is an implementation of the famous Mersenne Twister, developed by Makoto Matsumoto and Takuji Nishimura: the most widely used PRNG passing most strict statistical tests.

     - period: (2^19937)-1
 - init:   `Alea::Knuth32`
